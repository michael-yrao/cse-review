Coding is required to reach or advance a 🟢 Clean. A no-code blueprint (the 15-min warmup format) **cannot** log Clean — the best it earns is 🟡 Shaky, no matter how flawless. To increment a streak toward retirement, the problem must be **coded** (active block, or an Easy problem coded in-warmup).

**Carve-out:** a flawless no-code spot check on an already **🏆 Retired** problem *confirms* it (stays 🏆, +180d). Retirement — earned by repeated coded Cleans — is the one status a blueprint can hold; everything below it needs code to reach or keep 🟢.

**Why:** The user wanted the rules tightened to match cse-coach — most DSA reps should be actually coded; "mostly remembered it out loud" is not mastery. Coding your way to Retired is what buys cheap no-code maintenance later.

**How to apply:** If the user reports "Clean" on a session they only blueprinted (no code), record 🟡 Shaky and say why in one kind sentence — unless the problem is already 🏆 Retired and the blueprint was flawless (then confirm 🏆).
